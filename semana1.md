# Semana 1 - Fase 1

## Lo que aprendí
- Instalé Ubuntu Server 24.04 en VirtualBox
- Me conecté por SSH desde Windows
- Aprendí comandos básicos de Linux
- Aprendí permisos con chmod
- Aprendí a controlar servicios con systemctl
- Instalé Git y conecté con GitHub

## Comandos importantes
- ip addr show → ver IP del servidor
- ssh usuario@ip → conectarse al servidor
- chmod 600/644/755 → cambiar permisos
- sudo systemctl start/stop/status → controlar servicios
- git clone/add/commit/push → control de versiones
